Descripción del proyecto
Este repositorio contiene la solución completa del famoso ejercicio práctico del curso "Databases and SQL for Data Science with Python" de IBM en Coursera (Módulo 5 – Peer-graded Assignment).
El objetivo es demostrar el dominio de consultas SQL avanzadas utilizando un dataset real de lanzamientos de SpaceX (Falcon 9) para responder preguntas de negocio como:

¿Qué sitios de lanzamiento existen?
¿Cuál es la carga útil promedio por versión del cohete?
¿Cuándo se logró el primer aterrizaje exitoso en plataforma terrestre?
¿Qué boosters han transportado la máxima carga útil?
